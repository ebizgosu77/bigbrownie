LESSON 01
변수와 함수 — "입력→출력 자판기"
★☆☆ 입문
수식에서 x, y, w, b 같은 문자가 나오면 당황하는 분들을 위해. 문자는 "변할 수 있는 숫자의 이름표"일 뿐입니다.
개념변수(Variable)란?
🎒 비유: "오늘 온도"를 x라 부르기로 약속한 것.
어제 x=5, 오늘 x=18 — x는 값을 담는 상자입니다.
개념함수 f(x)란?
🏭 비유: 함수는 자판기. 동전(x)을 넣으면 → 음료(y)가 나옴.
f(x) = 2x+1 이면, x=3 → y=7 출력.
입력 x=3
→
f(x)=2x+1
→
출력 y=7
🤖
AI 연결: ChatGPT도 결국 "다음 단어 예측 함수"입니다.
입력(텍스트) → 수백만 개의 f(x) → 출력(다음 단어)

SECTION 01

딥러닝·머신러닝 핵심 수식과 AI 활용 예시

선형대수 · Linear Algebra
\( \mathbf{y} = W\mathbf{x} + \mathbf{b} \)
선형 변환 (행렬 곱)
신경망의 모든 레이어는 입력 벡터 x에 가중치 행렬 W를 곱하고 편향 b를 더하는 선형 변환입니다.
📌 AI 활용: 완전연결층(FC Layer), 임베딩 레이어, Transformer의 Q·K·V 행렬 연산 전부 이 공식입니다.
미적분 · Calculus
\( \frac{\partial L}{\partial W} = \frac{\partial L}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial W} \)
역전파 (Chain Rule)
손실 함수 L을 가중치 W로 편미분하여 기울기를 계산합니다. 연쇄 법칙으로 레이어를 역방향으로 통과하며 모든 파라미터를 업데이트합니다.
📌 AI 활용: Backpropagation — 딥러닝 학습의 근간. PyTorch의 .backward()가 이 계산을 자동으로 수행합니다.
최적화 · Optimization
\( \theta \leftarrow \theta - \alpha \nabla_\theta L(\theta) \)
경사 하강법 (Gradient Descent)
학습률 α 방향으로 손실 기울기 ∇L만큼 파라미터를 업데이트합니다. Adam, RMSProp 등 모든 옵티마이저의 기본 원리입니다.
📌 AI 활용: 모든 ML/DL 모델 훈련 — SGD, Adam, AdaGrad 전부 이 원리의 변형입니다.
확률·통계 · Probability
\( P(y|x) = \frac{P(x|y)\,P(y)}{P(x)} \)
베이즈 정리 (Bayes' Theorem)
사전 확률 P(y)와 우도 P(x|y)를 결합해 사후 확률 P(y|x)를 구합니다. 불확실성을 수치로 표현하는 핵심 공식입니다.
📌 AI 활용: Naive Bayes 분류기, 베이지안 최적화, VAE의 잠재 분포 추정, LLM의 불확실성 추정.
정보이론 · Information Theory
\( \mathcal{L} = -\sum_{c} y_c \log(\hat{y}_c) \)
크로스 엔트로피 손실
실제 레이블 y와 예측 확률 ŷ 사이의 차이를 정보이론적으로 측정합니다. 분류 문제에서 가장 많이 사용되는 손실 함수입니다.
📌 AI 활용: 이미지 분류, GPT의 다음 토큰 예측 학습, BERT의 MLM(Masked Language Model) 사전학습.
선형대수 · SVD / PCA
\( A = U \Sigma V^\top \)
특이값 분해 (SVD)
행렬 A를 세 행렬의 곱으로 분해합니다. 데이터의 주요 성분(방향)을 추출하고, 차원 축소·노이즈 제거·추천 시스템에 활용됩니다.
📌 AI 활용: PCA 차원 축소, 협업 필터링 추천 시스템, LoRA(LLM 경량화)의 핵심 수학 원리.

사전 미션 개념 문제

Q.01★☆☆ 기초
벡터 내적과 유사도
두 벡터 a=(3,4)와 b=(1,2)의 내적을 구하고, 코사인 유사도를 계산하세요.
\( \text{cos sim}(\mathbf{a},\mathbf{b}) = \frac{\mathbf{a}\cdot\mathbf{b}}{\|\mathbf{a}\|\|\mathbf{b}\|} \)
①내적: 3×1 + 4×2 = 11
②∥a∥=5, ∥b∥=√5 → cos sim = 11/(5√5) ≈ 0.984
💡1에 가까울수록 의미가 유사한 단어/문장임을 뜻합니다.
Q.02★☆☆ 기초
행렬 곱셈 — FC Layer 시뮬레이션
입력 벡터 x=[1,2]ᵀ, 가중치 행렬 W=[[2,0],[1,3]], 편향 b=[1,−1]ᵀ일 때, 선형 변환 y=Wx+b를 계산하세요.
\( \mathbf{y} = W\mathbf{x} + \mathbf{b} \)
①Wx = [2×1+0×2, 1×1+3×2] = [2, 7]
②y = [2+1, 7−1] = [3, 6]
💡신경망 한 레이어가 바로 이 연산. 이어서 활성화 함수를 적용합니다.
Q.03★★☆ 중급
편미분과 경사 하강법
손실 함수 L(w)=(w−3)²가 있을 때, w=1에서의 미분값을 구하고, α=0.1로 한 스텝 경사 하강법을 적용한 뒤 새 w 값을 구하세요.
\( w \leftarrow w - \alpha \frac{dL}{dw} \)
①dL/dw = 2(w−3), w=1 → −4
②w_new = 1 − 0.1×(−4) = 1.4
💡w가 최솟값(3) 방향으로 이동했습니다. 이것이 AI 학습의 원리입니다.
Q.04★★☆ 중급
연쇄 법칙 — 역전파 1단계
z=w·x, ŷ=σ(z)=1/(1+e⁻ᶻ), L=(ŷ−y)² 일 때, x=2, w=0.5, y=1로 두고 ∂L/∂w를 연쇄 법칙으로 계산하세요.
\( \frac{\partial L}{\partial w} = \frac{\partial L}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial z} \cdot \frac{\partial z}{\partial w} \)
①z=1, ŷ=σ(1)≈0.731
②∂L/∂ŷ = 2(0.731−1) = −0.538
③∂ŷ/∂z = 0.731×0.269 ≈ 0.197, ∂z/∂w = 2
④∂L/∂w = −0.538×0.197×2 ≈ −0.212
Q.05★★☆ 중급
베이즈 정리 적용
스팸 메일 판별기. 전체 메일 중 스팸 비율 P(S)=0.3. 스팸 메일에 "무료"가 포함될 확률 P(무료|S)=0.8. 일반 메일에서 "무료" 포함 확률 P(무료|¬S)=0.1. "무료"가 포함된 메일이 스팸일 확률을 구하세요.
\( P(S|\text{무료}) = \frac{P(\text{무료}|S)\,P(S)}{P(\text{무료})} \)
①P(무료) = 0.8×0.3 + 0.1×0.7 = 0.24 + 0.07 = 0.31
②P(S|무료) = (0.8×0.3) / 0.31 = 0.24/0.31 ≈ 0.774
💡"무료"가 있으면 약 77.4% 확률로 스팸입니다. 나이브 베이즈 분류기의 원리입니다.
Q.06★★☆ 중급
크로스 엔트로피 손실 계산
3-class 분류 문제에서 정답 레이블 y=[1,0,0]이고, 모델 예측 ŷ=[0.7,0.2,0.1]일 때 크로스 엔트로피 손실을 계산하세요.
\( \mathcal{L} = -\sum_{c} y_c \log(\hat{y}_c) \)
①L = −(1×log(0.7) + 0×log(0.2) + 0×log(0.1))
②L = −log(0.7) ≈ −(−0.357) = 0.357
💡정답 클래스의 예측 확률만 손실에 영향. 확률 높을수록 손실↓
Q.07★★★ 심화
MSE와 그래디언트
MSE 손실 L=(1/n)Σᵢ(yᵢ−ŷᵢ)²에서 데이터 [y₁=2, y₂=4], 예측 [ŷ₁=1, ŷ₂=3]일 때 MSE를 계산하고, ŷ₁에 대한 ∂L/∂ŷ₁ 를 구하세요.
\( \frac{\partial L}{\partial \hat{y}_i} = -\frac{2}{n}(y_i - \hat{y}_i) \)
①MSE = (1/2)×[(2−1)² + (4−3)²] = (1/2)×2 = 1.0
②∂L/∂ŷ₁ = −(2/2)×(2−1) = −1×1 = −1.0
💡기울기가 음수 → ŷ₁을 증가 방향으로 업데이트해야 손실↓
Q.08★★★ 심화
SVD와 차원 축소
행렬 A의 SVD A=UΣVᵀ에서 Σ=diag(5,3,1)일 때, 상위 2개 특이값만 사용하여 rank-2 근사를 만들 때 보존되는 정보 비율(%)을 구하세요.
\( A_k = U_k \Sigma_k V_k^\top \)
①전체 에너지 = 5²+3²+1² = 25+9+1 = 35
②rank-2 에너지 = 5²+3² = 25+9 = 34
③보존 비율 = 34/35 ≈ 97.1%
💡PCA·LoRA의 핵심 원리. 상위 특이값 몇 개만으로도 대부분의 정보를 보존합니다.
